# 🖥️ Personal Portfolio & Observability CV Website

Welcome to my personal portfolio repository! This website is built using raw HTML, modern responsive CSS (with glassmorphic elements), and vanilla JavaScript. It is hosted on GitHub Pages and showcases my career details, skills, and projects as a Python Developer & Observability Engineer.

👉 **Live Site**: [https://jhonssonc.github.io/](https://jhonssonc.github.io/)

---

## 🌟 Key Features

* **Bilingual Translation (EN/ES)**: Seamless, client-side language toggling between English and Spanish without page reloads.
* **Interactive Observability Dashboard**: A simulated monitoring panel (inspired by Datadog & AppDynamics) illustrating:
  * Dynamic, live-fluctuating system metrics (Uptime, Latency, CPU Load).
  * Mock APM Traces checking endpoint performance.
  * Real-time RPA execution logs illustrating automated workflows.
* **Fully Responsive Design**: Styled to look premium on desktops, tablets, and smartphones.
* **Downloadable resumes**: Direct download links to professionally typeset Word versions (`.docx`) in English and Spanish.

---

## 🛠️ Technology Stack

* **Structure**: HTML5 (Semantic elements)
* **Styling**: Modern CSS3 (Variables, flexbox, CSS Grid, Glassmorphism, animations)
* **Logic**: Vanilla JavaScript ES6
* **Icons**: FontAwesome 6
* **Hosting**: GitHub Pages (`.nojekyll` pipeline)

---

## 📂 Repository Structure

* `index.html`: Main page containing all bilingual content and structure.
* `style.css`: Layout, design styles, and responsiveness.
* `script.js`: State management, translations, and dashboard simulations.
* `Jhonsson_Cordova_CV_English.docx` / `Jhonsson_Cordova_CV_Espanol.docx`: Downloadable CV artifacts.
